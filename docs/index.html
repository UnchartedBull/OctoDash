About

OctoDash is an Open-Source Dashboard for OctoPrint, which is optimized for smaller touchscreen devices. I mainly created this project, because there was just no nice looking UI available, which could be ran on the Raspberry Pi to provide access to OctoPrint where it is needed most - directly at the printer. The main goal for OctoDash is to offer a "lite" experience of OctoPrint in a very compelling and easy-to-use design. It is important to note here, that OctoDash does not replace OctoPrint, but rather uses the backend of said awesome program and just offers a different User Interface.

I currently develop OctoDash, with the help from a lot of other people over on GitHub in my freetime. If you want to support OctoDash you can do that, by either creating PRs to fix open issues or donate via the Sponsor button in GitHub, if you really like it :).

As mentioned above, OctoDash is completely Open-Source and all the Source Code can be found in the GitHub. The whole project is licensed under the Apache 2.0 License, so go ahead and create your own custom version! If you think you've done something really cool open up a PR and it may get merged into master. But please, keep the LICENSE file and Copyright notices unchanged, if you want to remove those contact me first please.

I'm hoping that you're going to enjoy OctoDash!

Timon

Installation

The installation script, which can be found here, will work for installing and updating OctoDash. If a new version is available, just execute the script again and it will install the latest available version.
There also is a manual installation guide available in the GitHub Wiki.
